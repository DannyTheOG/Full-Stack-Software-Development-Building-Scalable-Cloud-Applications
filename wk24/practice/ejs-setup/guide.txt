run 'npm i ejs' to install ejs

add to your index.js:
    app.set( 'views', path.join( process.cwd(), 'views' ) );
    app.set( 'view engine', 'ejs' )

create .ejs file in the views folder

use res.render() to serve the rendered HTML along with dynamic data

use <%=%> tag to render the data in HTML( this takes in any JS expression, including plain variables )
