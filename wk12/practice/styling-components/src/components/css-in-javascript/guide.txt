To run the CSS-in-JS libraries example, we first have to install the latest version of the "Styled Components" package using npm or yarn


1. stop the current react app execution( running ) in your terminal using Ctrl + C keys

2. install the package using the following command on your terminal:
    {{ npm install styled-component@latest }}

3. start the react app execution again in your terminal using the command:
    {{ npm start }}
