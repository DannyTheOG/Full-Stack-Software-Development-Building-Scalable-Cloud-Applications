1. Download the zip folder and extract the week4-app folder.

2. Inside the week4-app folder, we have 4 different folders representing four topics of this week.

3. Also, each folder in itself is a React App.

4. Open a terminal and navigate to the respective folder.

    For example, for running the counter-rtk app, navigate to the inner counter-rtk folder in the VSCode terminal.

5. Run the following command to install all dependencies-

npm install

6. Execute the app using the following command-

npm start
